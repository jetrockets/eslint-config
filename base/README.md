## Getting Started

A package for basic linter configuration of javascript code

1. Install package
   ```sh
   npm install @jetrockets/eslint-config-base@latest
   ```
2. Enter your API in `.eslintrc.js`
   ```js
   module.exports = {
     extends: [
      '@jetrockets/eslint-config-base',
      ...
    ],
    ...
   }
   ```
